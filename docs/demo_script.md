# Demo Script (5–7 min)

1) Toggle laten zien (Good→Bad) en benoemen wat verandert (copy, velden, uitleg).
2) Formulier invullen met 2–3 scenario's (laag inkomen + hoge risicohouding).
3) In Good-mode: uitleg (feature‑bijdragen) en onzekerheidsmelding (0.45–0.55).
4) In Bad-mode: geen uitleg, nudge/urgency, extra velden (BSN).
5) Data‑download tonen (portabiliteit).
6) Afsluiten met reflectie op rubric: interactie, ethiek, AI‑UX.
